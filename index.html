Odin Project 3

Welcome to the Rock-Paper-Scissors game!

Instructions and Reminders:

  1. Go to the Console tab. Press Command+Option+J (Mac) or Control+Shift+J (Windows, Linux, ChromeOS).
  2. Enter game(); in the console. Alert and prompt pop-up windows will appear.
  3. Keep in mind that the only options are rock, paper and scissors. Make sure you spell these correctly!
  4. There are 5 rounds in each game and the score reverts to 0 once a game ends.
  5. Repeat Step 2 if you want to continue playing after the game ends. Enjoy!
